# Talliffy Brand Icon

Place the provided Talliffy app icon image in this folder with the filename:

- `talliffy-icon.png`

Recommended specs:
- Format: PNG
- Size: 128×128 px or larger (will be downscaled in UI)
- Background: Keep as-is (do not alter colors)

Usage in app:
- Collapsed sidebar: 28×28 px
- Expanded sidebar: 22×22 px

Paths referenced:
- `assets/brand/talliffy-icon.png`

If you prefer SVG, keep the same filename (`talliffy-icon.svg`) and I can update the references accordingly.
